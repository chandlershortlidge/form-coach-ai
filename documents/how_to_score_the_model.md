# Accuracy vs precision scoring method 

False positive: the model confidently presents something as good advice when it's actually bad advice. 

False negative: the model not mentioning something important from the reference — like forgetting to mention wrist position entirely. 

High precision: low false positives
High accuracy: low false negatives

We want to emphasis precision: if the model forgets some good advice, that is better than the model giving bad (potentially harmful) advice.
